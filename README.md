# AI Model Training Project

This repository provides a basic structure for training AI models. It includes directories for source code, data, and Jupyter notebooks, as well as a minimal example for getting started.

## Structure
- `src/` - Source code for model training
- `data/` - Datasets and data files
- `notebooks/` - Jupyter notebooks for experiments

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run the example script: `python src/train_example.py`

See `requirements.txt` for dependencies.
